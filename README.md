# モンスターファーム2 技の使い込みトラッカー

モンスターファーム2の技の使い込み進捗を管理するWebアプリです。

## 機能

- 全38種族の使い込み技データを収録
- 大会ごとの使い込み回数を記録
- 合格 / やり直し（破棄）で累計を管理
- ガッツ回復速度に基づく理想使用回数を自動計算
- データはブラウザに自動保存
- PWA対応（スマホのホーム画面に追加可能）
- ダークモード対応

## GitHub Pagesで公開する方法

1. このリポジトリをGitHubにpush
2. Settings → Pages → Source: `main` ブランチ、`/ (root)` を選択
3. 保存すると `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開される

## スマホのホーム画面に追加する方法

### iPhone (Safari)
1. Safariでアプリを開く
2. 共有ボタン → 「ホーム画面に追加」

### Android (Chrome)
1. Chromeでアプリを開く
2. メニュー → 「ホーム画面に追加」

## ファイル構成

```
index.html     # メインアプリ
manifest.json  # PWA設定
sw.js          # Service Worker（オフライン対応）
icon-192.svg   # アイコン（192x192）
icon-512.svg   # アイコン（512x512）
README.md      # このファイル
```

## データについて

技データは有志Wikiを参照して作成しています。  
ガッツ回復速度は「30ガッツ回復にかかる秒数」で入力してください（6〜19）。
